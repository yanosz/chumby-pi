chumby-pi apt repository

Signed flat apt repository for chumby-pi — the Chumby Classic control panel on a Raspberry Pi (arm64, Debian trixie / Raspberry Pi OS).

Install

curl -fsSL https://yanosz.github.io/chumby-pi/chumby-archive.gpg \
  | sudo tee /usr/share/keyrings/chumby-archive.gpg >/dev/null
echo 'deb [signed-by=/usr/share/keyrings/chumby-archive.gpg] https://yanosz.github.io/chumby-pi ./' \
  | sudo tee /etc/apt/sources.list.d/chumby.list
sudo apt update
sudo apt install chumby-player

The package contains no Chumby firmware. On first start the player tells you which files to copy from your chumby (or its backup) into /var/lib/chumby — controlpanel.swf is required; widgets, alarm tones and the intro are optional.
